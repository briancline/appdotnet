stream(stream_url, decode=True)[source]¶

A generator used to iterate over events received via the streaming API. Automatic JSON decoding is the default behavior but can be disabled.

Yields an Event instance for each line received via the API. If decoding is off, yields a string of the raw line received.

Parameters:
  • stream_url – the stream endpoint URL
  • decode (boolean) – whether to decode the JSON structure before yielding.
Returns:

(appdotnet.api.Event) yields an Event instance for each streaming API event received
